How It Works

1. Data Collection

Satellite images (350×350px) from MapBox API using wildfire coordinates from Canada's Open Government Portal

2. CNN Processing

Convolutional Neural Network extracts spatial features: vegetation density, heat signatures, terrain patterns

3. MLP Classification

Multilayer Perceptron combines features to classify risk as High, Medium, or Low with confidence scores

4. Real-time Alerts

Instant predictions enable early warnings, resource allocation, and prevention strategies
